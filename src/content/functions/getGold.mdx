---
id: getGold
name: getGold
description: Returns the amount of gold a player currently has.
syntax: $getGold($player)
parameters:
  - name: $player
    type: string
    required: true
    description: The player to get gold from
returns: The amount of gold player currently has
examples:
  - code: /chat @a $getGold($player) /g
    description: Display player's gold in chat
lang: en
---

Returns the amount of gold a player currently has. Use this function to check a player's gold balance or display it in commands.
