variables:
  - name: playerId
    type: number
    description: The ID of the player
  - name: player
    type: string
    description: The username of the player
  - name: previous
    type: number
    description: The previous hunger value
  - name: current
    type: number
    description: The current hunger value
  - name: delta
    type: number

Triggered when a player's hunger level changes. Useful for hunger monitoring, starvation warnings, or food systems.
