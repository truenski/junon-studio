---
id: role
name: /role
description: Assign a role to a player.
syntax:
  - /role [player] [rolename]
category: with-args
lang: en
args:
  - name: player
    type: string
    required: true
    description: The username of the player
  - name: rolename
    type: string
    required: true
    description: The name of the role to assign
examples:
  - code: /role PlayerName admin
    description: Assign the "admin" role to PlayerName
  - code: /role PlayerName moderator
    description: Assign the "moderator" role to PlayerName
---

Assigns roles to players, which can be used for permissions and access control.
